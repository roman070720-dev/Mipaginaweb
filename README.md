## ✨ Características

* **Diseño Responsivo:** Adaptable a móviles, tabletas y computadoras.
* **Modo Oscuro/Claro:** Interfaz personalizable para el usuario.
* **Interactividad:** Rendimiento rápido y transiciones suaves.
* **Optimización SEO:** Estructura limpia para buscadores.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica del sitio.
* **CSS3:** Estilos, animaciones y diseño responsivo (Flexbox/Grid).
* **JavaScript (ES6):** Lógica del lado del cliente e interactividad.
* *[Opcional]* **Frameworks:** Bootstrap / Tailwind CSS / React (elimina si no los usas).

## 🚀 Instalación y Uso Local

Sigue estos pasos para ejecutar el proyecto en tu computadora:

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com
   ```
2. **Navega al directorio del proyecto:**
   ```bash
   cd tu-repositorio
   ```
3. **Abre el proyecto:**
   * Haz doble clic en el archivo `index.html`.
   * O usa la extensión **Live Server** en VS Code.
